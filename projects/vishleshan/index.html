Vishleshan

An MVC framework for Conversion Optimization Platforms.

Many a times, ecommerce websites, CRO Platforms, or any other informatic website requires a lot of development work for basic SQL queries, JS code execution on front end and other information's display over the platform. Vishleshan is a framework that would provide you an abstraction of all the development work required to execute frame JS Code and SQL queries on your website. So, for instance, you need a JS Code for location on your website: "_abc.com_location('a')" or something like this. So, you need to frame this code as well in order to execute it. It has all the capability of adding third party vendors like Shopify to your platform. Vishleshan will frame the code for you. See detailed report for more in-depth knowledge.
